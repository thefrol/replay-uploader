# Установка

создайте файл `.env` следующего содержания

    BUCKET=highlights-uploads
    AWS_ACCESS_KEY_ID=... # тут ключ
    AWS_SECRET_ACCESS_KEY=... # тут ключ
    FIFO=incoming-videos.fifo

установите пакеты

    pip install -r requirements.txt
